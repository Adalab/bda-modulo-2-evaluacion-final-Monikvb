🔍**`Consultas SQL en BBDD 'SAKILA'`**🔍

En el siguiente repositorio encontrará un código de SQL con ¿25? preguntas(Consultas).

En ella, podemos ver tablas como:
- film (películas) 
- actor (actores) 
- customer (clientes) 
- rental (alquileres)
- category (categorías)
- inventory (inventario)
